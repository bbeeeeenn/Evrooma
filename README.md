# EVROOMA

## A Web-Based Classroom Availability Monitoring System

The purpose of this system is to provide real-time, accurate, and accessible classroom availability information to students, instructors, and administrators.

---

Evrooma is a standalone web application that enables:

- Real-time monitoring of classroom occupancy
- QR-based room usage updates by instructors
- Dashboard access for students, instructors, and administrators
- Report generation for classroom utilization
